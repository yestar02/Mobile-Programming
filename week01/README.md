## 학습 목표
- `if / when`을 이용해 메뉴 선택을 분기 처리한다.
- `while` 또는 `do-while`로 메뉴를 반복 실행한다.
- `readln()` 입력을 안전하게 처리한다 (`toIntOrNull()` 활용).

## 수업에서 만들 프로그램(요구사항)
아래 형태의 메뉴를 출력하고, 사용자의 입력(숫자)을 받아 동작하도록 구현합니다.

예시 UI(콘솔):
===== Smart Task & Note =====
1. 항목 추가
2. 전체 목록 보기
3. 키워드 검색
4. 종료
메뉴 선택:


### 필수 기능
- 메뉴를 반복해서 보여준다.
- 숫자가 아닌 입력/빈 입력이 들어와도 프로그램이 종료되지 않는다.
- 4번(종료)을 선택하면 정상 종료된다.

### 기능 확장
- “항목 추가” 기능(문자열 1줄 입력 받아 저장)
- “전체 목록 보기” 기능
- “키워드 검색” 기능(입력한 단어가 포함된 항목만 출력)

> 참고: 1주차는 컬렉션을 아직 배우지 않았으므로 Java의 저장 방식을 사용합니다.
> - `Array<String?>` + index로 간단 저장

## 실행 방법 (Android Studio)
> 이번 주는 **콘솔 프로젝트**입니다. Android 앱(Compose) 프로젝트가 아닙니다.

1) Android Studio → **New Project**
2) **Kotlin/JVM(또는 Kotlin)** 기반 템플릿(콘솔) 선택  
3) 프로젝트 생성 후 `main()` 함수가 있는 파일을 실행합니다.
   - 실행 버튼(▶) 또는 `main()` 옆 Run 아이콘 사용

### 실행 확인 체크
- 실행했을 때 메뉴가 뜨는지
- 잘못 입력해도(문자) 종료되지 않는지
- 4번 종료가 정상 동작하는지
