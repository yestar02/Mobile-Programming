## 📖 핵심 기능 설명

### 1번 기능: 데이터 모델 정의 (`DressingItem`)

1) 관련 디렉터리 및 파일: `data/model/DressingItem.kt`

2) 설명
   - 체크박스와 인형 이미지의 상태를 관리하기 위해 데이터 클래스로 구성
   - **`name (String)`**: 체크박스 옆에 나타날 부위의 텍스트
   - **`id (Int)`**: 주어진 이미지들을 `res/drawable`에 넣었고, 해당 이미지들을 불러오기 위한 변수
   - **`isSelected (Boolean)`**: 각 부위의 체크박스 선택 여부를 저장하는 변수, 기본값 `false`로 하여 아무것도 선택되있지 않도록 구성
   - **`drawLevel (Int)`**: **이미지가 그려질 순서에 관한 변수**. 안경, 모자, 콧수염은 눈, 눈썹, 입보다 위에 그려져야 하기 때문에 이를 정렬하기 위한 변수.

3) 점검 사항 및 설명
